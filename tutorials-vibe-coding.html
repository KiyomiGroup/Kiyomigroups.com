What is Vibe Coding?

Coined by Andrej Karpathy. Named Word of the Year 2025.

Vibe Coding was coined by Andrej Karpathy — OpenAI co-founder and Tesla AI director — in February 2025. Collins English Dictionary named it their Word of the Year 2025.

The idea is simple: instead of writing every line of code yourself, you describe what you want to build in plain English. AI tools like Cursor, Lovable, and Claude generate the code. You review, guide, and ship.

You become the architect — not the bricklayer. This course teaches you exactly how to do that, from first prompt to deployed product.

8-Week Curriculum

Everything you need. Nothing you don't.

Module 1 — The Vibe Coding Mindset

Week 1

What vibe coding is, why it works, and how to think like a product director instead of a programmer. History, tools landscape, and the AI-first workflow.

Module 2 — Your AI Toolkit

Week 2

Deep dives into Cursor, Lovable, Replit, GitHub Copilot, Claude, and ChatGPT. When to use each tool, how to configure them, and how to combine them for maximum output.

Module 3 — Prompt Engineering for Code

Week 3

The art of writing prompts that actually work. Context windows, chaining prompts, iterating on AI output, and knowing when to debug vs re-prompt.

Module 4 — Build Your First App

Weeks 4–5

Full walkthrough: idea → wireframe → AI-generated code → deployed product. You'll ship a real, live web app by the end of this module. No shortcuts.

Module 5 — Vibe Coding with Kiyomi's Stack

Week 6

Apply vibe coding to production-grade tools: Ruby on Rails, React, and Elixir. How Kiyomi's real engineering team integrates AI into regulated, compliance-heavy workflows.

Module 6 — Security, Quality & the Pitfalls

Week 7

AI code has known risks — security gaps, logic errors, and hallucinated dependencies. Learn how to review, test, and validate AI-generated code like a senior engineer would.

Module 7 — Capstone: Ship Your Product

Week 8

Build and deploy your capstone project from scratch using everything you've learned. Peer review, instructor feedback, and certificate of completion upon passing.
